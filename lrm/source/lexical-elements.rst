Lexical Elements
================

The following Ada reserved words should not be used in |SPARK|:
**abort**,
**accept**,
**access**,
**aliased**,
**delay**,
**entry**,
**exception**,
**goto**,
**protected**,
**raise**,
**requeue**,
**select**,
**synchronized**,
**task**,
**terminate**,
**until**.
