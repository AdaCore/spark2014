Example of illegal code:

```ada
procedure Example with SPARK_Mode => On is
   type U (D : Boolean := True) is record
      case D is
         when True  => X : Integer;
         when False => Y : Float;
      end case;
   end record with Unchecked_Union;

   Obj : U with Relaxed_Initialization;  -- error
begin
   null;
end Example;
```

This error is issued when a part of an `Unchecked_Union` type is given relaxed
initialization. [Section 6.10(8) of the SPARK Reference Manual] states that no
part of an `Unchecked_Union` type shall have relaxed initialization. As in Ada,
the parts of an object include the object as a whole, so annotating an object of
an `Unchecked_Union` type, as with `Obj` here, is itself a violation.

The fix is to remove the `Relaxed_Initialization` aspect; the object then follows
the standard data initialization policy:

```ada
procedure Example with SPARK_Mode => On is
   type U (D : Boolean := True) is record
      case D is
         when True  => X : Integer;
         when False => Y : Float;
      end case;
   end record with Unchecked_Union;

   Obj : U;
begin
   Obj := (D => True, X => 0);
end Example;
```

Relaxed initialization cannot be combined with an `Unchecked_Union` type. Any
data that must be exempted from the standard initialization policy has to be
held in a type that is not an `Unchecked_Union`.

[Section 6.10(8) of the SPARK Reference Manual]:
https://docs.adacore.com/live/wave/spark2014/html/spark2014_rm/subprograms.html#relaxed-initialization
