Example of illegal code:

```ada
package Example with SPARK_Mode => On is
   type Root is tagged record
      Value : Integer;
   end record;

   procedure Op (X : Root);

   type Root_Proc is access procedure (X : Root);

   Cb : Root_Proc := Op'Access; -- error
end Example;
```

This error is issued on an `'Access` attribute reference whose prefix is a
dispatching operation (a primitive of a tagged type). [Section 4.1.4(2) of the
SPARK Reference Manual] forbids taking `'Access` of a dispatching operation.

The fix is to use a non-dispatching subprogram instead. A subprogram declared
in a package body after the tagged type's freeze point is not a primitive and
therefore not dispatching:

```ada
package Example with SPARK_Mode => On is
   type Root is tagged record
      Value : Integer;
   end record;

   procedure Op (X : in out Root);

   type Root_Proc is access procedure (X : in out Root);
end Example;
```

```ada
package body Example with SPARK_Mode => On is
   procedure Op (X : in out Root) is
   begin
      X.Value := X.Value + 1;
   end Op;

   procedure Op_Non_Dispatch (X : in out Root) is
   begin
      X.Value := X.Value + 1;
   end Op_Non_Dispatch;

   Cb : Root_Proc := Op_Non_Dispatch'Access;
end Example;
```

Because `Op_Non_Dispatch` is declared in the package body, it is not a
primitive of `Root` and can be used with `'Access`.

[Section 4.1.4(2) of the SPARK Reference Manual]:
https://docs.adacore.com/live/wave/spark2014/html/spark2014_rm/names-and-expressions.html#attributes
