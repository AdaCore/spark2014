Example of illegal code:

```ada
package P with SPARK_Mode => On is

   Sensor : Natural with Volatile, Async_Writers;

   type Container is new Natural
     with Iterable =>
       (First       => First,  -- error: First is named in an Iterable aspect
        Next        => Next,   --        and is a volatile function
        Has_Element => Has_Element);

   function First (X : Container) return Natural
     with Volatile_Function,
          Global => (Input => Sensor);
   function Next        (X : Container; Y : Natural) return Natural is (Y + 1);
   function Has_Element (X : Container; Y : Natural) return Boolean
     is (Y < Natural (X));

end P;
```

This error is issued on a function named in the `Iterable` aspect of a
container type (i.e., one of `First`, `Next`, `Last`, `Previous`,
`Has_Element`, `Element`, or `Constant_Reference`) when that function is
declared with the `Volatile_Function` aspect. A volatile function in SPARK
is a function declared `with Volatile_Function`; it is permitted to read
volatile inputs (such as variables with `Async_Writers`), but in exchange
the language restricts where calls to it may appear, because each call may
observe a different value of the underlying volatile state.

[Section 5.5.2(12) of the SPARK Reference Manual] forbids this, currently.

The fix is to declare the iteration primitive as an ordinary
(non-volatile) function. If the iteration genuinely needs to depend on
volatile state, read that state into a non-volatile container value
outside the loop and iterate over the snapshot:

```ada
package P with SPARK_Mode => On is

   Sensor : Natural with Volatile, Async_Writers;

   type Container is new Natural
     with Iterable => (First       => First,
                       Next        => Next,
                       Has_Element => Has_Element);

   function First       (X : Container) return Natural is (0);
   function Next        (X : Container; Y : Natural) return Natural
     is (Y + 1);
   function Has_Element (X : Container; Y : Natural) return Boolean
     is (Y < Natural (X));

   procedure Snapshot (C : out Container)
     with Global => (Input => Sensor);

end P;
```

```ada
package body P with SPARK_Mode => On is

   procedure Snapshot (C : out Container) is
   begin
      C := Container (Sensor);
   end Snapshot;

end P;
```

[Section 5.5.2(12) of the SPARK Reference Manual]:
